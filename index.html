The Quiz Manager

Why the Quiz Manager? In this day and age, across many industries it has become vital to make sure that your employees are in compliance with federal, state, and company policy. As part of the compliance, tests or readings are often required.

Quiz Manager is an application that allows admins to monitor their list of users and the statuses of the users’ required materials. It also creates a platform for users to view the list of required materials and access the content. This was a one week project, where a partner and I were tasked to create an application using React/Redux. My contributions to this application was working on the front end.

To log into the demo as an admin use the username: Admin and password: test-password. To log in as a user use the username: UserOne and password: test-password.

Project Stack: React, Redux, Node, Express, Mongoose, MongoDB
